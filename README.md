# Nextcloud Deployment on Ubuntu

This project demonstrates the deployment of a self-hosted Nextcloud server on Ubuntu using the LAMP stack.

## Stack

- Ubuntu Server
- Apache2
- MySQL
- PHP
- Nextcloud

## Steps

1. Installed Apache web server
2. Installed MySQL and created database
3. Installed PHP and required modules
4. Downloaded and deployed Nextcloud
5. Configured permissions for Apache
6. Connected Nextcloud to MySQL database
7. Completed web installation

## Result

Working self-hosted Nextcloud instance accessible via browser.

## Purpose

This project was created as a homelab exercise to practice Linux server administration and DevOps fundamentals.
